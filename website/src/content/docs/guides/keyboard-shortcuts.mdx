import { Kbd } from 'starlight-kbd/components'

These work anywhere in the application

| Shortcut                | Action                   | Description                                             |
| ----------------------- | ------------------------ | ------------------------------------------------------- |
| <Kbd mac="Command+S" /> | Save File                | Save the currently open file                            |
| <Kbd mac="Command+N" /> | New File                 | Create a new file in the selected collection            |
| <Kbd mac="Command+W" /> | Close File               | Close the currently open file                           |
| <Kbd mac="Command+K" /> | Command Palette          | Open the command palette to search and execute commands |
| <Kbd mac="Command+," /> | Preferences              | Open application preferences                            |
| <Kbd mac="Command+0" /> | Focus Editor             | Focus the main editor from anywhere in the app          |
| <Kbd mac="Command+1" /> | Toggle Sidebar           | Show/hide the left sidebar (file browser)               |
| <Kbd mac="Command+2" /> | Toggle Frontmatter Panel | Show/hide the right sidebar (frontmatter editor)        |

## Component Builder Keyboard Shortcuts

When the Component Builder dialog is open:

| Shortcut                | Action           | Description                                                |
| ----------------------- | ---------------- | ---------------------------------------------------------- |
| <Kbd mac="Delete" />    | Go Back          | Return to component selection (when in configuration step) |
| <Kbd mac="Command+A" /> | Toggle All Props | Select/deselect all optional component properties          |
| <Kbd mac="Return" />    | Insert Component | Insert the configured component into the editor            |
