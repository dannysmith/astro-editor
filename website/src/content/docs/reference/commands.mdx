---
title: 'Commands'
description: 'Reference for the commands available in the Astro Editor command palette'
---

Open the command palette with `Cmd+P` from anywhere in the app, then type to fuzzy-search — for example, typing "foc" finds "Toggle Focus Mode". See [The Editor](/editor/overview/#the-command-palette) for an overview.

Some commands only appear when they're relevant — file commands need an open file, IDE commands need a [configured IDE](/file-management/ide-integration/), and there's one "Open Collection" command per collection in your project.

## File

| Command | Description |
| --- | --- |
| **New File** | Create a new file in the selected collection |
| **Close File** | Close the current file |

## Navigation

| Command | Description |
| --- | --- |
| **Toggle Sidebar** | Show or hide the sidebar |
| **Toggle Frontmatter Panel** | Show or hide the frontmatter panel |
| **Find Content** | Search content to open a file or insert a link (the [Content Linker](/editor/links/#content-linker)) |
| **Open Collection: _name_** | Switch to a different collection (one command per collection) |

## Project

| Command | Description |
| --- | --- |
| **Open Project** | Select a new project folder |
| **Reload Collections** | Refresh the project structure |

## Settings & Modes

| Command | Description |
| --- | --- |
| **Open Preferences** | Open application preferences and settings |
| **Toggle Focus Mode** | Dim all text except the current sentence |
| **Toggle Typewriter Mode** | Keep the cursor line centred in the viewport |

## Copyedit Highlights

These toggle the [copyedit highlighting modes](/editor/copyedit-modes/).

| Command | Description |
| --- | --- |
| **Toggle Nouns / Verbs / Adjectives / Adverbs / Conjunctions** | Toggle highlighting for that part of speech |
| **Show / Hide All Highlights** | Toggle all part-of-speech highlights together |

## IDE Integration

These require an [IDE command](/file-management/ide-integration/) to be configured.

| Command | Description |
| --- | --- |
| **Open Project in IDE** | Open the current project in your preferred IDE |
| **Open Collection in IDE** | Open the current collection directory in your preferred IDE |
| **Open File in IDE** | Open the current file in your preferred IDE |
