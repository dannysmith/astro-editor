---
title: 'Opening a Project'
description: 'How to open an Astro project and configure common path overrides'
---

Use `File > Open Project` to select your Astro project directory. Astro Editor will automatically scan for content collections defined in your `src/content/config.ts` file.

## Directory Restrictions

For security reasons, Astro Editor cannot open projects located in certain system directories. If you attempt to open a project in one of these locations, you'll see an error message asking you to choose a different location.
