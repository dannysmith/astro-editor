## Updating

Astro Editor keeps itself up to date. It checks for a new version shortly after launch, and you can check at any time from **Check for Updates…** in the application menu.

When a newer version is available, an **Update Available** dialog shows the version and its release notes, and lets you:

- **Install it** — the update downloads, then Astro Editor offers to restart to apply it.
- **Skip This Version** — you won't be prompted about that version again automatically (a manual check still shows it).
- **Remind Me Later** — dismiss the prompt for now.

If you installed with Homebrew, you can also update from the terminal with `brew upgrade --cask astro-editor`.
