Astro Editor is available for macOS, Windows, and Linux.See the [Releases](/releases/) page for release notes and previous versions.

## macOS

<LinkButton href="https://github.com/dannysmith/astro-editor/releases/latest/download/astro-editor-latest.dmg" icon="cloud-download" iconPlacement="start">
  Download for macOS (.dmg)
</LinkButton>

Or install with [Homebrew](https://brew.sh/):

```bash
brew install --cask astro-editor
```
