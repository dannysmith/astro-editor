The editor is a **markdown source editor**, not a preview. You always see the real markdown, but the syntax characters (the `#`, `*`, `` ` `` and so on) are dimmed so they stay out of your way, while the text they affect is styled to match. A few common edits also have keyboard shortcuts — see [Keyboard Shortcuts](/reference/keyboard-shortcuts/) for the complete list.

## Headings

Start a line with one to six `#` characters. The `#` marks are dimmed, and when there's room they "hang" in the left margin so your heading text stays aligned with the body. Heading colour is configurable in [Preferences](/preferences/).

```markdown
# Heading 1
## Heading 2
### Heading 3
```

Rather than typing the `#` yourself, you can transform the current line:

| Shortcut | Action |
| --- | --- |
| `Opt+Cmd+1`–`Opt+Cmd+4` | Set the line to a Heading 1–4 |
| `Opt+Cmd+0` | Remove the heading (back to a paragraph) |

## Bold & Italic

```markdown
**bold text** and *italic text*
```

| Shortcut | Action |
| --- | --- |
| `Cmd+B` | Wrap the selection in `**` (bold) |
| `Cmd+I` | Wrap the selection in `*` (italic) |

Both shortcuts toggle — run them again on the same selection to remove the formatting.

## Lists

Unordered lists start with `-`, `*`, or `+`; ordered lists start with a number followed by a dot. Just start typing — there's no shortcut needed.

```markdown
- First item
- Second item

1. First step
2. Second step
```

## Links

Write links with standard markdown, or use `Cmd+K` to wrap the selected text in a link and drop your cursor into the URL. Hold `Opt` and click any link to open it in your browser.

```markdown
[Astro Editor](https://astroeditor.danny.is)
```

To link to another item in your content collections, use the Content Linker (`Cmd+Shift+K`) — see [Links](/editor/links/) for more.

## Inline Code & Code Blocks

Wrap inline code in single backticks; it's shown in Astro Editor's monospace font with a subtle background. Fenced code blocks (three backticks) get a tinted background across the whole block.

````markdown
Use the `glob()` loader in your config.

```ts
const articles = defineCollection({ /* ... */ })
```
````

Code **inside** fenced blocks is deliberately not syntax-highlighted by language — Astro Editor is a writing tool, not a coding one. See the [Philosophy](/getting-started/philosophy/) page for why.

## Blockquotes

Start a line with `>`. The whole quoted line is shown in grey italics.

```markdown
> A quote stands out from the surrounding text.
```

## Other Syntax

The editor also styles images (`![alt](src)` — hold `Opt` and hover to [preview them](/editor/images-and-files/)), horizontal rules (`---`), and inline HTML.
