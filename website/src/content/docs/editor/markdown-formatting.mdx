## Headings

Convert the current line to a heading with `Opt+Cmd+1` through `Opt+Cmd+4`, or back to a plain paragraph with `Opt+Cmd+0`. See the [Keyboard Shortcuts](/reference/keyboard-shortcuts/) reference for the full list of editor shortcuts.
