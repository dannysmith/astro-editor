## Distraction-Free Writing

Hide either sidebar with <Kbd mac="Cmd+1" windows="Ctrl+1" /> (left) and <Kbd mac="Cmd+2" windows="Ctrl+2" /> (right). With both closed, the editor fills the window for a clean, distraction-free writing surface. Focus and Typewriter modes build on this.

## Focus Mode

Dims everything but the current sentence (or line for lists). Can be toggled with the 👁️ icon in the toolbar, with <Kbd mac="Cmd+Shift+F" windows="Ctrl+Shift+F" />, or via the command palette.

## Typewriter Mode

Typewriter mode keeps the cursor line vertically centered in the editor at all times. As you type or navigate, the viewport scrolls to keep the cursor in the middle of the visible area, creating an immersive writing experience similar to iA Writer's typewriter mode.

Toggle it with <Kbd mac="Cmd+Shift+T" windows="Ctrl+Shift+T" />, or via the command palette ("Toggle Typewriter Mode").

Typewriter mode works well alongside Focus Mode — enable both for a deeply focused writing experience where your current sentence stays front and center.

**Note**: Typewriter mode is session-only and resets when you restart the app.
