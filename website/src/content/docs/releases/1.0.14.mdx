Not many user-facing changes, but a few under-the-hood changes.

- **Beta Support for deb and rpm packages** — the app now ships as .deb, .rpm, and AppImage, all with in-app auto-update.
- **MDX component builder** — scanning a component path outside the project root now returns no results instead of surfacing an error.
