## Opening in Your IDE

The "Open in IDE" command launches your preferred code editor with either the current file or the entire project. Configure your IDE command in preferences (<Kbd mac="Cmd+," windows="Ctrl+," />). Popular options include:

- `code` for Visual Studio Code
- `cursor` for Cursor
- `subl` for Sublime Text
